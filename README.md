# NVIM

## Installation

- nvim: `sudo dnf install neovim`
- ctags: `sudo dnf install ctags`

## coc-extensions

    node has to be installed

    - coc-clangd
    - coc-docker
    - coc-eslint
    - coc-explorer
    - coc-fzf-preview
    - coc-html
    - coc-java
    - coc-json
    - coc-markdownlint
    - coc-marketplace
    - coc-omnisharp
    - coc-snippets
    - coc-sql

    - coc-tsserver
